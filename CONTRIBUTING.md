# Contributing

Thank you for your interest in these Siril scripts!

## Asking Questions

The best place to ask questions — including questions about using `.md` files with Claude Code or any other topic related to this repository — is to **open a GitHub Issue** in this repository:

👉 [Open an Issue](../../issues/new)

GitHub Issues is the primary support channel for this project. Please include as much relevant context as possible in your question.

## Reporting Bugs

If you find a bug, please open an issue and include:

- A description of the problem
- Steps to reproduce it
- The Siril version you are using
- The operating system you are using

## Submitting Changes

1. Fork the repository.
2. Create a branch for your change.
3. Make your changes and test them with Siril.
4. Open a Pull Request describing what you changed and why.

## Using Claude Code with this Repository

[Claude Code](https://docs.anthropic.com/en/docs/claude-code) is Anthropic's AI coding assistant. When you open this repository in Claude Code, it automatically reads `CLAUDE.md` to understand the project context.

If you have questions about using Claude Code or about the `CLAUDE.md` file in this repository, feel free to open a GitHub Issue.
